1. System maszynowy

Ta cecha wymaga, aby system był oparty na technologii komputerowej, zautomatyzowany i skalowalny.
Wszystkie podcechy muszą być spełnione (TAK), aby cecha była uznana za spełnioną.

Podcecha 1: Czy system wykorzystuje oprogramowanie do przetwarzania danych (np. algorytmy, modele ML)?



Podcecha 2: Czy system działa na dedykowanym sprzęcie komputerowym (np. serwery, procesory GPU, chipy wbudowane)?



Podcecha 3: Czy system jest zintegrowany z innymi technologiami (np. aplikacje, urządzenia IoT)?



Podcecha 4: Czy system przetwarza dane w sposób zautomatyzowany za pomocą kodu lub algorytmów?



Podcecha 5: Czy system jest oparty na technologiach umożliwiających skalowalność (np. chmura, rozproszone przetwarzanie)?



2. Różny poziom autonomii

Ta cecha wymaga, aby system działał samodzielnie w pewnym stopniu, z warunkiem, że przynajmniej 3 z 5 podcech muszą być spełnione (TAK).

Podcecha 1: Czy system przetwarza dane i generuje wyniki bez ciągłego nadzoru człowieka?



Podcecha 2: Czy system wykonuje zadania samodzielnie w określonych scenariuszach, nawet przy minimalnym nadzorze?



Podcecha 3: Czy system podejmuje decyzje lub wykonuje działania bez ręcznego wprowadzania każdej instrukcji?



Podcecha 4: Czy system może działać w sposób ciągły bez interwencji użytkownika po początkowej konfiguracji?



Podcecha 5: Czy system reaguje automatycznie na zmiany w danych wejściowych lub środowisku?



3. Zdolność adaptacji

Ta cecha wymaga, aby system mógł się uczyć i dostosowywać, z warunkiem, że przynajmniej 2 z 5 podcech muszą być spełnione (TAK).

Podcecha 1: Czy system uczy się lub aktualizuje swoje modele na podstawie nowych danych (np. uczenie maszynowe)?



Podcecha 2: Czy system dostosowuje swoje działanie do preferencji użytkownika lub kontekstu (np. personalizacja)?



Podcecha 3: Czy system zmienia swoje zachowanie bez konieczności ręcznej modyfikacji kodu?



Podcecha 4: Czy system optymalizuje swoje działanie na podstawie wcześniejszych wyników lub doświadczeń?



Podcecha 5: Czy system może dostosowywać się do nowych typów danych lub zadań po wdrożeniu?



4. Wnioskowanie

Ta cecha wymaga, aby system generował wyniki na podstawie danych, z warunkiem, że wszystkie podcechy muszą być spełnione (TAK).

Podcecha 1: Czy system analizuje dane wejściowe w celu generowania wyników (np. klasyfikacja, predykcja)?



Podcecha 2: Czy system generuje predykcje, decyzje, treści lub zalecenia na podstawie danych?



Podcecha 3: Czy system wyciąga wnioski na podstawie celów zdefiniowanych przez twórcę lub dorozumianych?



Podcecha 4: Czy system przetwarza dane w sposób, który umożliwia identyfikację wzorców lub anomalii?



Podcecha 5: Czy system dostarcza wyniki, które są zależne od złożonego przetwarzania danych (np. nie tylko proste reguły)?



5. Wpływ na środowisko

Ta cecha wymaga, aby system miał wpływ na otoczenie, z warunkiem, że przynajmniej 2 z 5 podcech muszą być spełnione (TAK).

Podcecha 1: Czy system wpływa na decyzje lub zachowania użytkowników (np. rekomendacje, scoring)?



Podcecha 2: Czy system kontroluje lub oddziałuje na urządzenia w świecie fizycznym (np. roboty, pojazdy)?



Podcecha 3: Czy system zmienia treści, procesy lub interakcje w środowisku wirtualnym (np. media społecznościowe)?



Podcecha 4: Czy wyniki systemu mają potencjalne konsekwencje dla ludzi, organizacji lub innych systemów?



Podcecha 5: Czy system wpływa na alokację zasobów lub wyniki procesów (np. decyzje kredytowe, rekrutacja)?
